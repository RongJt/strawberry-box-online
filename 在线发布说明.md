# 在线发布说明

`strawberry_box_online_share` 是给别人手机编辑用的网页发布目录。

## 发布后别人能做什么

- 手机打开网页。
- 载入推荐初始布局。
- 拖动贴片、补字、复制裁剪。
- 导出回传 JSON 发给你。

## 推荐发布方式

可以用任意静态网页托管：

- GitHub Pages
- Netlify
- Cloudflare Pages
- Vercel
- 自己的服务器

上传整个 `strawberry_box_online_share` 目录即可。

## 重要

- 不要只发本地 `127.0.0.1` 地址，别人打不开。
- 不要只发压缩包让对方手机直接打开，手机浏览器通常无法正常读取压缩包内的素材。
- 对方回传给你的应该是 JSON，不是 40 张 PNG。

## 你收到回传文件后

在桌面版 `index.html` 点“导入贴字布局”，选择对方回传的 JSON。
