# 发给别人手机编辑并回传

最推荐的方式是把本项目放到一个网页地址上，然后把链接发给别人。别人不需要安装软件，只要用手机浏览器打开链接即可编辑。

## 对方怎么编辑

1. 用手机打开你发的网页链接。
2. 进入 `手机编辑.html`。
3. 点“载入初始布局”。
4. 拖动贴片、补字、复制裁剪。
5. 编辑完点“导出回传文件”。
6. 把导出的 JSON 发回给你。

## 你怎么接收

1. 打开自己的桌面版 `index.html`。
2. 点“导入贴字布局”。
3. 选择对方发回来的 JSON。
4. 检查修改效果。
5. 需要成图时，导出 40 页 ZIP 或运行 `render_layout.ps1`。

## 为什么只回传 JSON

回传 JSON 很小，只记录文字贴片的位置、裁剪、补字内容和样式。你不需要让对方回传 40 张大图。

## 发布给别人用

目录 `strawberry_box_online_share` 是准备好的网页发布版。把这个目录上传到 GitHub Pages、Netlify、Cloudflare Pages、个人服务器或网盘网页托管后，把网页链接发给别人。

如果只是发 ZIP 给别人，对方手机通常不能直接在压缩包里运行网页；最好还是给一个网页链接。
